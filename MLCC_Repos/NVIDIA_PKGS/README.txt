
In this directory, you should download NVIDIA's cuDNN from https://developer.nvidia.com/cudnn
(NVIDIA requires registration in their Developer Program to access cuDNN).

For whichever CUDA versions you will use, MLCC expects files in this directory named similar to these:

- cudnn-10.0-linux-x64-v7.4.2.24.tgz
- cudnn-9.2-linux-x64-v7.4.2.24.tgz
- cudnn-9.1-linux-x64-v7.1.tgz
- cudnn-9.0-linux-x64-v7.4.2.24.tgz
- cudnn-8.0-linux-x64-v7.1.tgz
